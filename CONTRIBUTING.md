# Contributing

Contributions should make public AI-native company research clearer, more useful, and better sourced.

## Good Contributions

- Add a new source note with links and original analysis.
- Improve a pattern with concrete implementation steps.
- Add a caution or failure mode.
- Convert an abstract idea into a checklist.
- Correct source metadata or overstatement.

## Required Format For Source Notes

```markdown
# Source Title

Source: <URL>
Publisher:
Speaker / Author:
Published:
Accessed:

## Summary

## Key Ideas

## Analysis

## Practical Applications

## Risks / Open Questions
```

## Review Checklist

- No full transcript or copied long-form text.
- No private data.
- Source URL included.
- Summary and analysis are separated.
- Claims are not stronger than the evidence.
- Practical advice has explicit boundaries.
